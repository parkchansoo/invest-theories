# Second-Level Thinking — NVDA DeepSeek 진입 사례

> 박찬수의 진짜 알파를 가장 깨끗하게 보여준 매매 결정.

## 하워드 막스의 Second-Level Thinking

`Howard Marks`의 *The Most Important Thing*에서 정의:

- **1차적 사고 (First-level thinking)**: 컨센서스 결론. 누구나 도달 가능. 가격에 이미 반영.
- **2차적 사고 (Second-level thinking)**: 컨센서스 너머의 결론. 같은 정보에서 남들과 다른 결론을 도출. 알파의 원천.

> *"First-level thinking says, 'It's a good company; let's buy the stock.' Second-level thinking says, 'It's a good company, but everyone thinks it's a great company, and it's not. So the stock's overrated and overpriced; let's sell.'"* — Howard Marks

알파를 만들려면 컨센서스가 틀렸을 때 정확히 그 자리에 있어야 한다.

## 박찬수의 사례 — 2025-01-27 DeepSeek 충격

### 시장 상황

2025년 1월 27일, 중국 AI 스타트업 DeepSeek이 R1 모델 공개. NVIDIA H100보다 10배 적은 자원으로 GPT-4 수준 성능 달성 보고. **NVDA 하루에 -17% 폭락 ($142.62 → $118.42)**.

### 1차적 사고 (시장 컨센서스)

```
DeepSeek이 효율적 학습 → 같은 성능에 GPU 적게 필요
→ AI 기업들 capex 줄일 수 있음
→ NVDA 매출 둔화
→ NVDA 매도
```

이 논리로 시장 전체가 NVDA를 던졌고, 가격이 -17% 빠졌다.

### 2차적 사고 (박찬수)

박찬수 본인 표현:
> "DeepSeek 하락이 매수기회였다고 생각. **아무리 효율이 좋아져도 AI 에 대한 수요는 그 이상**일 거라고 생각했어."

논리 구조:
```
DeepSeek이 효율적 학습 → 같은 비용에 더 많은 AI 학습 가능
→ AI 보급 가속 (소형 기업·개인까지 AI 학습 가능)
→ AI 사용 사례 폭발적 증가
→ 추론(inference) 수요 폭증 → GPU 수요 증가
→ NVDA 매수
```

**같은 사실(효율 향상)에서 정반대 결론**.

### 결과 검증

| 시점 | NVDA | 시장 narrative |
|------|------|---------------|
| 2025-01-27 (DeepSeek 충격) | $118 | 1차적 사고 정점 |
| 2025-02-25 (박찬수 진입) | $126 | 회복 시작 |
| 2025-04-04 (관세 충격 저점) | $94 | 추가 매도 압력 |
| 2025-09-30 | $186 | 회복 + 신고가 부근 |
| **2026-04-25** | **$208** | **+76% from DeepSeek 저점** |

→ 박찬수의 2차적 사고가 정확. **컨센서스가 6개월 만에 박찬수의 결론으로 수렴**.

## 왜 이게 박찬수의 강점인가

### 1) 박찬수가 가진 정보 우위

- 개발자 직업 → AI/반도체 기술의 1차적 이해가 일반 투자자보다 깊음
- 하드웨어 효율 vs 소프트웨어 보급의 비대칭 효과(Jevons paradox)를 직관적으로 인식 가능
- "기술 효율 향상 → 사용량 증가"는 컴퓨터/통신 산업에서 반복된 패턴

### 2) Jevons Paradox (제본스 역설)

19세기 영국 경제학자 William Stanley Jevons의 발견:
> 석탄 효율이 높아진 증기기관이 등장하면 석탄 사용량이 줄어들 거라 예상했지만, **오히려 석탄 사용량은 폭증했다**. 효율이 높아지면 비용이 낮아지고, 낮아진 비용이 새로운 사용처를 열어 총수요를 키운다.

**같은 원리가 AI 칩 수요에 적용됨**:
- 학습 효율 향상 → 학습 비용 하락 → 더 많은 기업/개인이 AI 학습 가능 → 칩 수요 증가
- 추론 효율 향상 → 추론 비용 하락 → AI 서비스의 단위 사용량 폭증 → 칩 수요 증가

### 3) 박찬수가 본 것을 컨센서스가 못 본 이유

- 시장은 단기 narrative에 반응 (NVDA 매출 단기 둔화 우려)
- Jevons paradox는 6~24개월 시차로 작동 — 시장이 이 시차를 내재화하기 전에는 매수 기회

## 체크리스트 — 다른 종목/상황에 적용

다음 체크리스트로 second-level thinking 기회를 발굴:

### 단계 1: 컨센서스 결론 명문화
- [ ] 시장이 지금 이 종목/이슈에서 도달한 결론은 무엇인가?
- [ ] 그 결론이 가격에 어떻게 반영되어 있는가?

### 단계 2: 컨센서스의 전제 분리
- [ ] 그 결론을 위해 시장이 깔고 있는 전제 3개를 적어보기
- [ ] 각 전제가 정말 맞는가? 어느 전제가 가장 약한가?

### 단계 3: 정반대 시나리오 작성
- [ ] "이 전제가 틀린다면?" 시나리오 작성
- [ ] 그 시나리오가 실현되면 가격은 어디로 가야 하는가?

### 단계 4: 비대칭 보상 평가
- [ ] 컨센서스가 맞을 확률 vs 정반대 시나리오 확률
- [ ] 두 시나리오에서 손실 / 수익의 크기
- [ ] 비대칭 (기대값 양수)인가?

### 단계 5: 사이즈 결정
- [ ] 진입 사이즈는 1R 룰에 맞는가?
- [ ] 분할 매수 단계 설계 (가격 하락 시 추가 매수 룰)

## 자주 작동하는 second-level 패턴 (박찬수 관점)

### 패턴 1: 효율 향상 → 수요 증가 (Jevons paradox)
- AI 칩 효율 → AI 사용 폭증 → 칩 수요 증가 (NVDA 사례)
- 클라우드 효율 → 사용 폭증 → 인프라 수요 증가

### 패턴 2: 단기 충격 → 펀더멘털과 분리된 노이즈
- 전쟁 공포 → SK하이닉스 -10% 하락. **펀더멘털(HBM 수요)은 영향 없음** → 매수
- 일시적 매크로 우려 → 우량주 일시 하락 → 매수

### 패턴 3: 컨센서스가 형성된 후 → 추가 상승 한계
- "이미 다 좋다"가 컨센서스가 되면 → 작은 실망에도 큰 하락
- 주요 PER이 역사적 상위 5%면 매도 검토

### 패턴 4: 비주류 시나리오 (Tail opportunity)
- 시장이 무시하는 산업의 구조적 변화
- 5~10년 secular trend 위에 있지만 아직 컨센서스 형성 전
